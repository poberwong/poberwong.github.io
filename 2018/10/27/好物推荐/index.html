不是氪金党，但有一颗氪金的心

HHKB-BT

一个能够有效提升编码效率的神器，最开始对没有方向键的小键盘很有顾虑，就跟第一次使 Mac 是一样的。不过以经验来讲，任何不熟悉的东西，在拥有相关基础的情况下，磨合一到两周就可以了，尤其像键盘、电脑这种每天都长期接触的工具：

  1. 需要熟悉不同的键位，如最重要的 Command 键的排布，据说 HHKB 也是为了强化 Command 的可用性，所以才重排了它。
  2. HHKB 默认采取了 Fn 加上带有特殊功能键的方式来实现四个方向键以及其他功能键（屏幕亮度以及音量等），出于个人喜好，我采用了 Command + P + N + B + F + A + E 的方式来实现上下左右以及行首行尾（缺点，在 vim 模式下，上下键用组合键不太好用，应该是还没调整好）
  3. 反引号被放在了右上角，这样我们在同 App 之间切换窗口的时候只需要动动右手小拇指即可，不再让左手丢失键位去切换。

好了，能够尽最大可能减少双手重新寻找键位（FJ），被称为 Happy Hacking Keyboard 看来是有原因的。
价格 2200￥（官方价）

BOSE QC 35

以舒适降噪著称的 BOSE 在调研再三后成为了今年耳朵的主打歌。在使用了个把月后，给人的感觉就是保暖舒服加安静，现在十月份的深夜中打车回家的时候，听不到像车辆行驶、由远处逐渐衰减下来的各种低频噪音，静静的等待那辆铁壳四轮生物带你回家。
价格 2880￥ （官方指导价）

iPhone Xs

作为三年的果粉，被破万的苹果新机无数次吓到尿湿裤裆，可惜当初老罗发布会没能领到尿不湿 ~
真香！！！64G 不够用，256G 又太大，苹果可能就是这样，64G 单纯就是为了做陪跑的产品。在心疼钱包的同时，又为自己能够出门再也不用带充电宝、解放电脑磁盘、不用再忍受 6s 间歇性卡死、有大屏幕可以做更多事感到兴奋。
不过没有了 Apple Care+ 的护航，还是悠着点的好，千万不要用它来下水拍摄咯 ~ IP68 作为防尘防水等级最高的级别，苹果居然不能接受进水的事实 ……
价格 10099￥（大陆官方价）

Magic Mouse

不得不说，它是我最趁手的工具之一。本来不想提的，因为它的入手时间并非今年，已经是陪伴我三年的老同志了~ 但还是要说说。
它能够最大程度还原苹果自带触摸板的手势，能够让你在鼠标上同样能够感受到触摸板的快感 ！
价格 548￥（大陆官方价）

AirPods

这是继 MacBook Pro 之后让我如此亲睐的产品，首先在使用频率上来看，它是仅次于手机的存在，加之它卸下暂停、戴上自动连接或启动音乐、充电十五分钟使用两小时的存在，而且采用了上百人的耳朵做模型，它可想而知成为了手机音频无线化的最佳选择。总之给满分。
价格 1246（非无线充电版）
