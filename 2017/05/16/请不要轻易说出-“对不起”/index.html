请不要轻易说出 “对不起”

发表于 2017-05-16
随笔
首页 分类 标签 关于

曾经听过这么两句话：“在任何场合多说 ‘谢谢’ 总是没问题的”。“在犯错时，说 ‘谢谢你的帮助，我会做的更好’ 会比 ‘对不起’ 更让人感到舒服’”。这也是自己在很多陌生人眼里树立了一名彬彬有礼的学生形象。但与之相较，“对不起” 就显得在很多场合里格格不入了…

「对不起」一般是一种对对方致歉的最直接的表达方式。当两人在针对一个问题进行讨论时，如果出现因为一方的失误而出现一些不好的结果时，就需要这两种常用语来表达自身的态度。

  1. 对不起：
    当你在这种情况下通过这三个字表达你的态度时，会让对方感到极为不舒服。因为讨论是一个平等交流的过程，很容易因为个人情绪而导致一些比较激烈的争论，这是好事。因为每个人都认真对待这件事了，讨论才会变的有意义。此时如果你用一句「对不起」来给予回应，并不会产生任何积极的效果，却在很多情况下会让整个事情变得更加糟糕。对于你自己，可能算是给自己强行制造了一个台阶下，但是对于对方，你的一句「对不起」让整个交流失去了最基本的平衡性。会让对方觉得「卧槽，这还能不能讨论下去了，我到底做错了什么？算了，还是不讲了」，从而让你自己失去更多的交流和本应获取的信息，甚至失去这么一位 partner。

  2. 谢谢，我会做的更好：
    相对应，这句话会在讨论双方产生一个正催化的效果。当双方讨论到一定程度后，相对弱势的一方在讲出这句话后，另一方会感受到鼓舞和激励，但同时又回保持自己的言辞预期来保持进一步的交流，这会让对方觉得自己的讨论和努力室友价值有意义的，而非对对方造成了伤害。

讲了这么多，当然，对我而言，当碰到这种只会说 “对不起” 的人的时候，我会尽量去适应配合他，久而久之自然就 ……. 不理他了。哈哈哈哈～～～
